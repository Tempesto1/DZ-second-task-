# Паспорт курсового проекта

## Тема
Расписание для высших учебных заведений 

## Проблема
Проблема актуальности расписания и изменений в текущий момент времени

## Объект и предмет
Объект: Веб-приложение расписания 
Предмет: проектирование веб структуры

## Цель

Создать веб-приложение для автоматизации работы с расписанием вуза, позволяющее хранить данные о занятиях, группах, преподавателях и аудиториях, а также быстро обновлять расписание при изменениях.

## Основные задачи
1. Построить прототип интерфейса
2. Построить таблицу требований и uml
3. Оценить объём рынка таких же продуктов
4. Написать общую архитектуру “болванку” приложения

## Ожидаемый результат
Программа, методика, модель, анализ, макет, база данных, расчёт и т.д.
