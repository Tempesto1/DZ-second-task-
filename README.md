# Расписание для ВУЗов

## Назначение проекта

Данный проект поможет как студентам, так и преподавателям, эффективней использовать расписание, главная задача упрощение жизни студентов и преподавателей, с помощью различных технологий

## Состав репозитория
- docs/ — проектная документация и методика;
- src/ — исходные файлы, прототипы или скрипты;
- data/sample/ — небольшой пример данных;
- experiments/ — журнал проверок, расчётов или экспериментов;
- assets/ — изображения, схемы, диаграммы.

## План выполнения
1. Постановка задачи и анализ предметной области.
2. Подбор источников и требований.
3. Разработка метода, модели или прототипа.
4. Проверка результата.
5. Подготовка пояснительной записки.

## Текущий статус
сформирована постановка задачи, создана структура репозитория
## Статус проекта

Проект находится на этапе подготовки курсовой работы. В репозитории постепенно добавляются постановка задачи, описание методики, структура проекта и материалы для реализации веб-приложения расписания занятий в вузе.
